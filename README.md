Как работать с Git

Инициализация репозитория и первый коммит:
git init
git add training_planner.py
git commit -m "Initial: Training Planner with filtering and JSON export/import"
Игнорирование временных файлов и окружения:
Создайте файл .gitignore и добавьте: *.pyc, pycache/, venv/
Частые коммиты:
git add .
git commit -m "Добавлена функциональность планирования тренировок, фильтры и экспорт/импорт"
